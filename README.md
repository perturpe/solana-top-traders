# Solana Top Traders

## Features
- Track top traders on the Solana blockchain.
- Visualize trading stats and history.
- Get updates on token performance.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/perturpe/solana-top-traders.git
   cd solana-top-traders
   ```
2. Install necessary dependencies:
   ```bash
   npm install
   ```
3. Configure your environment variables as needed.
4. Run the application:
   ```bash
   npm start
   ```

## Usage Guide
- After running the application, you can access it via your web browser.
- Use the interface to navigate through trading stats and trades.

## Troubleshooting
- Ensure that your environment variables are set correctly.
- If you encounter issues, check the logs for errors.
- Common issues may relate to network connectivity. Verify your internet connection.
